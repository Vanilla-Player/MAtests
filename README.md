## Test hechos en clase

Este repo contiene los test que realizamos en clase, como parte tecnica

### Installation

```
git clone https://github.com/Vanilla-Player/MAtests.git
cd MAtests
npm install
```

### Tests

```
npm test ahorcado
npm cypress open
```
